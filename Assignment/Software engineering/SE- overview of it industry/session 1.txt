Question 1 : Name 5 IT job roles and write one line about what each role does.
Answer :
    Frontend developer - Developer designs and writes code for frontend design which is shown to the user/customer.
    Backend developer - Developer builds backend systems which run code on server and handles business logic, security, etc.
    Devops engineer - Developer creates CI/CD pipelines which automates integration and deployment of applications on server.
    Infrastructure engineer - engineer handles and manages servers and insfrastructure which runs all the appplications.
    Appplication tester - Tester tests and runs application which will be verfied if application is running according to requirements and approve for release.

Question 3 : Write the full forms and one-line use of: GitHub, Slack, Jira, IDE, API.
Answer :
    GitHub : used for code version control and sharing of code.
    Slack : slack is used for companies internel messaging and sharing files.
    Jira :  Used to track tickets and store important documnets and files.
    IDE : (Integrated development environment) environment which has all features used by developer such as, code editor, debugger, terminal etc.
    API : (Apllication programming interface) used to communicate between applications via any protocol such as restful apis, grpc, json etc.
