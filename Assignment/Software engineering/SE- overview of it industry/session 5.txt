Question 1 : Write the Git command for each: initialize a repo, stage all files, commit with a message, push to remote, pull latest changes.
Answer :
Initialize a repo : git init
Stage all files : git add .
Commit with a message : git commit -m "first commit"
Push to remote : git push origin main
Pull latest changes : git pull origin main

Question 2 : What is the difference between git fetch and git pull? Answer in 2 lines.
Answer :
git fetch downloads latest changes from remote repository but does not merge them.
git pull downloads latest changes and automatically merges them into current branch.

Question 4 : What is the difference between a Fork and a Pull Request? Answer in 2 lines.
Answer :
Fork creates a copy of another repository into your GitHub account.
Pull Request is used to request merging your changes into the original repository.
