Question 1 : Write the full form and one-line use of: HTTP, HTTPS, FTP, SMTP.
Answer :
HTTP : (Hyper Text Transfer Protocol) used for communication between browser and server.
HTTPS : (Hyper Text Transfer Protocol Secure) secure version of HTTP which encrypts the data.
FTP : (File Transfer Protocol) used to transfer files between systems over network.
SMTP : (Simple Mail Transfer Protocol) used for sending emails between mail servers.

Question 2 : What is the difference between IPv4 and IPv6? Answer in 2 lines.
Answer :
IPv4 uses 32 bit address format such as 192.168.1.1 and has limited number of addresses.
IPv6 uses 128 bit address format and provides very large number of unique IP addresses.

Question 4 : What do these HTTP status codes mean: 200, 301, 404, 500, 403?
Answer :
200 : Request completed successfully.
301 : Resource has been moved permanently to another URL.
404 : Requested page or resource was not found.
500 : Internal server error occured on backend server.
403 : Access to resource is forbidden due to permissions.

Question 5 : Explain DNS hierarchy (root, TLD, domain, subdomain) with [www.google.com](http://www.google.com) as an example.
Answer :
Root : highest level in DNS hierarchy represented by ".".
TLD : ".com" is the top level domain.
Domain : "google" is the domain name registered under .com.
Subdomain : "www" is subdomain of google.com used to access website.
